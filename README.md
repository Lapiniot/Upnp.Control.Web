## Upnp.Control.Web

Upnp.Control.Web is a modern web-based UPnP Control Point application focused on discovering, controlling, and monitoring multimedia devicesâ€”such as UPnP Media Servers and Media Renderers on your local network. It provides a Progressive Web App (PWA) user interface for seamless access across desktop and mobile devices.

### Key Features

- **UPnP Control Point for Multimedia Devices**: Automatically discovers UPnP Media Servers (for browsing and streaming content) and Media Renderers (for remote playback control).
- **Media Browsing and Playback**: Browse media libraries, select content, and control playback on networked audio/video devices directly from the web interface.
- **PWA User Interface**: The application features a responsive, installable Progressive Web App UI for a native-like experience.
- **Real-Time Updates**: Utilizes SignalR and push notifications for instant device status updates and event handling.
- **Extensible Architecture**: Built with modular .NET backend and a modern frontend, making it easy to add support for new device types and protocols.

### Use Cases

- Centralized control of networked multimedia devices
- Home media streaming and remote playback control
- Smart home automation and device management
- Monitoring and diagnostics for UPnP-enabled environments
